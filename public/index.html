Chekhov

What It Is

Chekhov is a project management tool. While a smart scheduling algorithm is under development, the application works as a simple case management system.

How It Works

The main elements are Clients, Cases, and Tasks. You can create new elements and associate them with each other. To make changes to existing elements, just edit inline and changes will be saved automatically.

What It's Made of

  • JavaScript, HTML, CSS frontend
  • Node.js server
  • MongoDB database
  • Heroku hosting
  • Travis CI
    • Mocha, Chai testing
